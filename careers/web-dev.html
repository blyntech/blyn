Web Developer

Location: Remote / Chennai
Job Type: Full-time
About Blyn Technologies

Blyn Technologies is a dynamic and innovative software development company specializing in web development, app development, and SaaS solutions. We empower businesses with cutting-edge technology, ensuring seamless digital experiences. Our mission is to create scalable, high-performing software solutions tailored to modern business needs.

Job Description

We are seeking a skilled Web Developer to join our team and contribute to the design, development, and maintenance of web applications. The ideal candidate is passionate about technology, has a keen eye for detail, and thrives in a fast-paced environment.

Responsibilities

  • Develop, test, and maintain responsive web applications using modern web technologies.
  • Collaborate with designers and backend developers to implement intuitive user interfaces (UI/UX).
  • Optimize web applications for speed, scalability, and performance.
  • Troubleshoot and debug issues to ensure smooth functionality.
  • Integrate third-party APIs, databases, and cloud services.
  • Stay up-to-date with industry trends and emerging technologies.
  • Participate in code reviews, documentation, and team discussions to ensure code quality and best practices.

Requirements

  • Proven experience in front-end and/or back-end web development.
  • Proficiency in HTML, CSS, JavaScript, and modern frameworks like React, Angular, or Vue.js.
  • Experience with backend technologies such as Node.js, Django, Laravel, or Express.js.
  • Familiarity with databases (MySQL, PostgreSQL, Firebase, MongoDB).
  • Understanding of RESTful APIs, authentication, and security best practices.
  • Experience with version control systems (Git, GitHub/GitLab).
  • Ability to work independently and in a collaborative team environment.

Nice-to-Have Skills

  • Experience with cloud platforms (AWS, Google Cloud, Firebase).
  • Knowledge of DevOps, CI/CD pipelines, and deployment strategies.
  • Familiarity with SEO principles and website optimization.

Perks & Benefits

  • Competitive salary based on experience.
  • Flexible work hours (Remote/Hybrid options).
  • Opportunities for career growth and skill development.
  • Work on exciting projects with a talented team.
  • Supportive and innovative work environment.
